### Getting Started 

#### Prerequisites
- Linux
- JDK 17
```
sudo apt install openjdk-17-jdk
```
if another jdk is installed on the system we can always select the one we want (17)
```
sudo update-alternatives --config java
java -version
```


Start the server:
```
./gradlew bootRun
```


Access server at: `http://localhost:8080`
